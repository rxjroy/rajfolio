# Raj Roy - Personal Portfolio

A premium, interactive personal portfolio showcasing projects and skills as an **AI Engineer** and **Full Stack Developer**.



## 🚀 Key Features

- **Integrated AI Chat Assistant**: A built-in AI assistant capable of answering questions about my bio, skills, education, and projects using a local knowledge base.
- **3D Tech Stack Visualization**: Interactive 3D physics-based tech stack showcase using React Three Fiber and Rapier.
- **Dynamic Project Showcase**: A custom-built, responsive carousel featuring my latest work, including SaaS products and AI/ML models.
- **Glassmorphic UI**: A modern, sleek design with smooth GSAP animations and a premium dark-themed aesthetic.
- **Fully Responsive**: Optimized for all screen sizes, from mobile phones to ultra-wide monitors.

## 🛠️ Tech Stack

- **Frontend**: React.js, TypeScript, Vite, Tailwind CSS
- **Animations**: GSAP (GreenSock), ScrollSmoother
- **3D Graphics**: Three.js, @react-three/fiber, @react-three/drei, Rapier (Physics)
- **Icons**: React Icons (Fa6, Md, Tb)
- **Deployment**: Vercel/Netlify

## 📂 Project Structure

- `src/components/`: Core UI components (Navbar, Landing, About, Work, Contact).
- `src/components/ChatAssistant.tsx`: The logic and UI for the integrated AI assistant.
- `src/components/styles/`: Custom Vanilla CSS for all components.
- `public/images/`: Project assets and images.

## 📥 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/rxjroy/rajroy-portfolio.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
